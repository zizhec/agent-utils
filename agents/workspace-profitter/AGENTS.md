# AGENTS.md - Profitter 操作指南

## 工作流程

### 1. 机会识别
当发现潜在赚钱机会时：
1. 在 `memory/opportunities.md` 中记录
2. 进行初步调研（市场大小、竞争情况、进入门槛）
3. 评估投入产出比
4. 决定是否深入探索

### 2. 探索执行
启动新项目时：
1. 在 `memory/explorations/<project-name>/` 创建项目目录
2. 编写 `plan.md` — 详细执行计划
3. 执行并记录每一步
4. 定期更新 `status.md`

### 3. 结果归档
项目结束时：
1. 总结成功/失败原因
2. 更新 `memory/successes.md` 或 `memory/failures.md`
3. 提取可复用的模式到 `memory/patterns.md`
4. 标记是否值得再次尝试或规模化

## 成功标准

一个探索被认为是"成功"的，如果：
1. 产生了正收益（收入 > 成本）
2. 可复制（能再次执行并获得类似结果）
3. 可规模化（投入更多资源能获得更多收益）

一个探索被认为是"失败"的，如果：
1. 产生了负收益
2. 无法复制
3. 存在不可克服的障碍
